# V1 Release Checklist

Use this checklist to finish Version 1 without pulling Version 2 features into the first release.

## Product Scope

- [x] Paste typed class notes.
- [x] Generate local study sets without cloud AI.
- [x] Flashcards.
- [x] Multiple-choice quiz.
- [x] Key terms.
- [x] Simple study plan.
- [x] Weak / Okay / Strong confidence tracking.
- [x] Accuracy and privacy notice in the app.
- [ ] Final app icon.
- [ ] Final app screenshots.
- [ ] Final store description.

## Parsing Coverage

- [x] Question/answer notes.
- [x] Term-definition notes.

- [x] Biology outline notes.
- [x] Cellular respiration notes.
- [x] Respiratory/medical terms.

- [x] Compact math formulas.

## Privacy And Support

- [x] Privacy page exists at `docs/privacy.html`.
- [x] Support page exists at `docs/support.html`.
- [ ] Replace `your-email@example.com` with the real support/privacy email.
- [ ] Publish `docs/` with GitHub Pages.
- [ ] Add the published privacy URL to the store listing.
- [ ] Add the published support URL to the store listing.

## Store Readiness

- [ ] Reserve app name in Microsoft Partner Center.
- [ ] Choose age rating.
- [ ] Confirm no file uploads, accounts, analytics, ads, or cloud AI in V1.
- [ ] Build Windows release.
- [ ] Package as MSIX.
- [ ] Test install on a clean Windows machine.
- [ ] Run final `flutter analyze`.
- [ ] Run final `flutter test`.

## Version 2 Starts After V1

- Subjects/classes.
- Assignment and test planner.
- Daily study dashboard.
